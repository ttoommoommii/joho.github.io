Codin’はマップ上の謎をプログラミングで解決し、ゴールを目指していく見下ろし型アクションパズルゲームです。マップ上に散らばっている「コード」を組み立てることで、行く手を阻むギミックを解決したり、敵の攻撃に対処することが出来ます。色とりどりのマス目はそれぞれが異なる意味を持ちます。そのマス目のギミックを利用しゴールまで進んでいくゲームです。
〇ゲーム説明（STARTをクリックするとゲームスタート）
　(1)マス目の種類と各ギミック
　　黒のマス　->　足場として作動していないマス
　　青のマス　->　普通の足場として作動しているマス
　　白のマス　->　ゴール、このマスの上に乗るとクリア
　　赤のマス　->　コードを使用することで通れるようになる
　　緑のマス　->　乗るともう片方の緑マスに瞬間移動
　　紫のマス　->　乗ると感染して回数制限が付く
　　水のマス　->　感染による回数制限の解除
　(2)操作方法（Wキーで上移動　Aキーで左移動　Sキーで下移動　Dキーで右移動　Fキーでコードを使用して　Enterキーで次へ進む）
　(3)GAMEOVERになる条件（敵に衝突、回数制限が０になる）
